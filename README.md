# 🏠 Real Estate Management System  

An end-to-end platform for efficient property buying and selling, built with **Java (Servlets/JSP)**, **MySQL**, and **JDBC**.  
The system provides secure role-based access for Admins, Agents, and Buyers with interactive web interfaces.  

---

## 🚀 Features  
- **Role-Based Access**  
  - **Admin**: Manage users, validate roles, audit activities  
  - **Agent**: Add, update, and remove property listings (CRUD operations)  
  - **Buyer/User**: Browse listings, filter/search properties, manage appointments  

- **Security & Integrity**  
  - JDBC `PreparedStatements` to prevent SQL Injection  
  - Strict role-based authorization  
  - Database transactions for consistency during updates  

- **Frontend**: Interactive UI with HTML, CSS, JavaScript  

---

## 🛠 Tech Stack  
- **Backend**: Java (Servlets/JSP)  
- **Database**: MySQL + JDBC  
- **Frontend**: HTML, CSS, JavaScript  

---

- Full-stack web development using Java  
- Secure application architecture with role-based access  
- Database management with MySQL & JDBC  
- Object-Oriented Programming concepts in practice
